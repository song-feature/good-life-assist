# 项目基本描述

- 我这里是一个生活助手 agent
- 这是一个前后端包含在一起的项目
- 它的前端样式基本上 左侧一个聊天框，右侧是一个功能区块，通过聊天对话
驱动 agent 加载对应功能的页面 并不断的展示相关功能


# 项目的基本功能

首先这是一个 agent智能体，可以不断的丰富技能和模块，
- 支持的技能模块要能通过配置管理页面来开启和关闭
- 每一个模块都会有单独的配置管理页面
- agent 模型要能够通过对话感知当前要使用的模块，并完成页面功能加载，
每一个模块都是一个子 agent，每一个子 agent 都可以有自己的模型配置


# 技术栈

- 服务端采用 python ，包管理采用 uv来管理
- agent 采用 react形式，使用langgraph和 langchain来实现
- 负责agent 驱动的大模型可以配置，默认使用 deepseek
- 有一个管理页面，可以管理所有的配置项
- 
- 前端采用 typescript
- 要求整个项目可以同时启动
